# Healthcare Management System

A comprehensive Java GUI-based application for managing hospital operations including patients, doctors, and appointments using PostgreSQL database.

## Features
- **Patient Management**: Add, view, edit, and delete patient records
- **Doctor Management**: Manage doctor profiles, specializations, and availability
- **Appointment Scheduling**: Schedule, view, and manage patient appointments
- **Medical Records**: Track diagnoses, prescriptions, and consultation fees
- **PostgreSQL Database**: Robust database persistence with proper relationships

## Technologies Used
- Java Swing (GUI)
- PostgreSQL Database
- JDBC Connectivity
- DAO Pattern
- MVC Architecture

## Prerequisites
- Java JDK 8 or higher
- PostgreSQL 12 or higher
- PostgreSQL JDBC Driver
